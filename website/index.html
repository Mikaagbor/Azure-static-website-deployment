Project Overview

This project demonstrates how a static website can be deployed to the cloud using Microsof Azure. The website files were created locally and then uploaded to Azure Blob Storage where static website hosting was enabled. Once deployed, the website became publicly accessible through a web URL.

  • Frontend: static website built with HTML, CSS and javascript
  • Cloud Platform: Microsoft Azure
  • Hosting Service: Azure Blob Storage static website hosting
  • Deployment Tool: Azure CLI
